# Trabalho 4 (T4) – Geração de Código I (Peso: 40% da M2)

### Apresentação em: 12/11/2025

#### Requisitos

Realizar a geração de código do programa usando o conjunto de instruções do processador BIP. O Código gerado deve executar corretamente no Bipide 3.0. As seguintes instruções devem ter seu código gerado:

- [x] (1 ponto) Declarações de variáveis e vetores (.data)
- [x] (1 ponto) Entrada de dados (com variáveis e vetores)
- [x] (1 ponto) Saída de dados (com inteiros, variáveis e vetores)
- [x] (3 ponto) Atribuições para variáveis e vetores
- [ ] (3 ponto) Operações aritméticas (soma e subtração e operações bit a bit com variáveis e vetores)
- [x] (1 ponto) Deverá haver na interface do editor um local para exibição do código assembly gerado (asm);

---

#### Observações: 

- [ ] Permitir operações dentro dos [ ] de um vetor, exemplo ```a[2+5]```
